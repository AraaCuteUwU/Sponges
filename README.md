# Sponges
A PocketMine-MP plugin that makes sponges working.

# How to use
* Install plugin [Dev Builds](https://poggit.pmmp.io/ci/AraaCuteUwU/Sponges/Sponges)
* Put the plugin into plugin folder
* Restart your server
* Enjoy the plugin
